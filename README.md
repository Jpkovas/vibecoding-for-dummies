# vibecoding-for-dummies skill repo

Repository ready for installation with `npx skills`.

## Structure

- `vibecoding-for-dummies/SKILL.md`
- `vibecoding-for-dummies/agents/openai.yaml`

## Installation via npx skills

After publishing this repository to GitHub:

```bash
npx skills add Jpkovas/vibecoding-for-dummies
```

To install directly without an interactive prompt:

```bash
npx skills add Jpkovas/vibecoding-for-dummies --skill vibecoding-for-dummies -y
```

## Quick verification

List skills available in the repository:

```bash
npx skills add Jpkovas/vibecoding-for-dummies --list
```
